# Love in Arabic Literature - Research Project Website

## Overview
A static research website about the evolution of love in Arabic-Islamic literature. The site presents academic research by Emna Ben Ameur under the supervision of Nicolas Baumard and Valentin Thouzeau (Université PSL, Institut Jean Nicod).

## Project Structure
```
.
├── server.py           # Python HTTP server for serving static files
├── site web/           # Static website files
│   ├── index.html      # Main HTML page
│   ├── styles.css      # CSS styling (garnet theme)
│   ├── app.js          # JavaScript (year display)
│   └── data/           # Data directory
└── README.md           # Project readme
```

## How to Run
The website is served using a simple Python HTTP server:
- Server: `python server.py`
- Port: 5000
- Host: 0.0.0.0

## Technical Details
- **Type**: Static HTML/CSS/JS website
- **Server**: Python http.server
- **Port**: 5000
- **Theme**: Bilingual (English/Arabic) academic research site with garnet color palette
